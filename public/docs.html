Developer Documentation

Welcome to the Brat Studio documentation. Here you'll find everything you need to know to use the web interface as a power user, as well as detailed API references for integrating Brat Studio into your own applications.

Getting Started

Brat Studio is a fully-featured, client-side canvas renderer backed by a lightweight Node.js API. It allows you to create static and dynamic text images featuring the iconic "brat" aesthetic.

Animation Modes

  • Normal: Instantly renders the text layout. Use for fast static outputs.
  • Animate (word): Sequentially reveals text word-by-word.
  • Typing (char): Replicates human typing with varied millisecond delays per character.

Adjust the Speed (WPM) slider to natively control the millisecond delays across all animations.

Themes & Backgrounds

Break away from the default white background by utilizing our advanced customization engine.

  • Opacity Masks: Upload any local `.jpg` or `.png` as a background. The app automatically applies a 45% opacity color-overlay so your text remains legible.
  • Object Fit: Adjust how your uploaded image fills the 512x512 canvas using `cover`, `contain`, or `stretch` algorithms.

Exporting Stickers

Brat Studio utilizes native browser APIs to give you instant exports without server-side processing delays.

  • Static (PNG): Triggers canvas.toDataURL() to download a highly requested transparent PNG.
  • Video (MP4/WebM): We utilize the built-in MediaRecorder API to capture a zero-latency real-time video stream of the <canvas> element. You can stop the recording manually at any time to capture the perfect loop.

API Reference

Our REST API allows you to programmatically create and share Brat posts. The API operates over standard HTTP methods and accepts/returns application/json payloads.

Base URL: https://justbratt.vercel.app/api

Authentication: No API key is required for the public endpoints.

Rate Limit: Each IP is limited to 60 requests/minute to prevent abuse.

Create a Post

Upload a new configuration to the database. You can supply different drawing modes (e.g. animate) and inject completely custom RGB/Hex color themes. Returns a unique 8-character ID used for sharing.

POST /api/create

Example Request: JavaScript fetch

const res = await fetch('https://justbratt.vercel.app/api/create', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    text: "brat and it's an api",
    mode: "animate",
    theme: JSON.stringify({ bg: "#ff0099", text: "#ffffff" })
  })
});
const data = await res.json();
console.log(data.id); // e.g., "e9b2f1a6"
  • text (required, string): The text to render (max 25 words).
  • mode (required, string): One of normal, animate, or typing.
  • theme (optional, string): A JSON stringified object containing bg and text hex color definitions to override defaults.

Response (200 OK)

{
  "id": "e9b2f1a6",
  "url": "https://justbratt.vercel.app/e9b2f1a6"
}

Bot Integration — Easy GET API

Specifically designed for WhatsApp, Discord, and Telegram bots. Use GET /api/create to save a post and get a shareable URL using just a simple HTTP GET request.

GET /api/create?text=...&mode=...
  • text (required): URL-encoded string.
  • mode (required): normal, animate, or typing.
  • theme (optional): URL-encoded JSON string e.g. %7B%22bg%22%3A%22%238ACE00%22%7D

Response (200 OK)

{
  "id": "e9b2f1a6",
  "url": "https://justbratt.vercel.app/e9b2f1a6"
}

🖼️ Sticker API — Direct PNG Image

The sticker endpoint renders and returns a real PNG image (512×512px) server-side. Just send this URL as a file in your bot — no frontend, no browser needed.

GET /api/sticker?text=...&bg=...&color=...

Query Parameters

  • text (required): The text to render. URL-encode it.
  • bg (optional): Background hex color. Default: #ffffff. Encode # as %23.
  • color (optional): Text hex color. Default: #000000.
  • imageUrl (optional): A public image URL to use as a background layer.
  • opacity (optional): Background image opacity 0–100. Default: 45.

Example URLs

# Default white/black
https://justbratt.vercel.app/api/sticker?text=hello+world

# Brat green background
https://justbratt.vercel.app/api/sticker?text=so+true+bestie&bg=%238ACE00

# Hot pink background, white text
https://justbratt.vercel.app/api/sticker?text=brat&bg=%23ff0099&color=%23ffffff

# With a background image from URL
https://justbratt.vercel.app/api/sticker?text=brat&imageUrl=https://example.com/photo.jpg&opacity=40

WhatsApp Bot (Baileys)

const fetch = require('node-fetch');

async function sendBratSticker(sock, jid, text, bg = '#8ACE00') {
    const url = `https://justbratt.vercel.app/api/sticker?` +
        `text=${encodeURIComponent(text)}&bg=${encodeURIComponent(bg)}`;

    const res = await fetch(url);
    const buffer = await res.buffer();

    await sock.sendMessage(jid, {
        sticker: buffer,
        mimetype: 'image/webp' // WhatsApp auto-converts PNG→WebP
    });
}

// Usage:
await sendBratSticker(sock, '6281234567890@s.whatsapp.net', 'brat and it\'s yours');

Discord Bot (discord.js)

const { AttachmentBuilder } = require('discord.js');
const fetch = require('node-fetch');

client.on('messageCreate', async (message) => {
    if (!message.content.startsWith('!brat ')) return;

    const text = message.content.slice(6).trim();
    const url = `https://justbratt.vercel.app/api/sticker?text=${encodeURIComponent(text)}&bg=%238ACE00`;

    const res = await fetch(url);
    const buffer = await res.buffer();
    const attachment = new AttachmentBuilder(buffer, { name: 'brat.png' });

    await message.reply({ files: [attachment] });
});

Telegram Bot (python-telegram-bot)

import requests
from telegram import Update
from telegram.ext import Application, CommandHandler, ContextTypes

async def brat(update: Update, ctx: ContextTypes.DEFAULT_TYPE):
    text = " ".join(ctx.args) if ctx.args else "brat"
    url = f"https://justbratt.vercel.app/api/sticker?text={requests.utils.quote(text)}&bg=%238ACE00"
    img_bytes = requests.get(url).content
    await update.message.reply_photo(photo=img_bytes, caption=f'"{text}" — JustBratt')

app = Application.builder().token("YOUR_BOT_TOKEN").build()
app.add_handler(CommandHandler("brat", brat))
app.run_polling()

Response

Returns a image/png binary (512×512px). Use it directly as a file attachment, sticker, or <img> src.

Retrieve a Post

Fetch the stored configuration of a specific post ID. This endpoint is utilized by the frontend to dynamically reconstruct shared links with the exact original animations and colors.

GET /api/posts/:id

Path Parameters

  • id (required, string): The unique 8-character hex ID returned by the Create endpoint.

Response (200 OK)

{
  "text": "it's brat but the api is animated and pink",
  "mode": "animate",
  "theme": "{\"bg\":\"#ff0099\",\"text\":\"#ffffff\"}"
}

Errors

  • 404 Not Found: Returned if the requested :id does not exist in the database.
  • 500 Internal Server Error: Returned if there is an SQLite read error.
